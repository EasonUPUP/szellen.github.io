AI-Mountain Car Problem

A group project solving the Mountain Car problem from OpenAI Gym environments. We attempt to solve the problem with 1) approximate Q-learning via simple neural network, and 2) exact Q-learning algorithm. We then analyze performance, training time, and ease-of-use of the two approaches.
View source code and paper

Food Recognition

This project was proposed as a calorie detection camera, using image recognition techniques to classify food and output corresponding calorie information. We used reduced-size Food 101 dataset as training data, applying transfer learning based on the pretrained AlexNet model to classify images.
